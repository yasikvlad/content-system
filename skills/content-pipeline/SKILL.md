# Контент-конвейер: из исходника в готовый контент (Notion-интеграция)

Этот skill автоматизирует полный цикл создания маркетингового контента из любого исходного материала и сохраняет всё в Notion.

## NOTION-КОНФИГУРАЦИЯ

Все данные сохраняются в базы на странице Content System. Создай ее для пользователя в Notion или удобном ему месте.

### Свойства «Исходники»:
- **Name** (title) — краткое название материала
- **Тип** (select) — Пост / Транскрипция / Статья / Заметка / Идея
- **Дата** (date) — дата добавления
- **Обработан** (checkbox) — обработан ли материал
- **Золотые куски** (number) — количество найденных золотых кусков
- **Заметки** (text) — доп. заметки
- Обратные relation-колонки «Контент RU» и «Content EN» заполняются автоматически

### Свойства « Контент »:
- **Название** (title) — название контент-единицы
- **Формат** (select) — Reels / Пост / Карусель / Тред / Лонгрид
- **Статус** (select) — Черновик / Готов / Выложен / В архиве
- **Выложено** (checkbox) — false при создании
- **Дата публикации** (date) — пустое при создании
- **Платформа** (multi_select) — Instagram / LinkedIn / Telegram / Twitter/X
- **Тема** (select) — AI/нейросети / Маркетинг / Инструменты / Кейсы / Личное
- **Исходник** (relation) — URL страницы из «Исходники»
- **Хронометраж** (select) — 15 сек / 20 сек / 25 сек / 45 сек / 55 сек / 60 сек
- **Кодовое слово** (rich_text) — байт для Telegram
- **Вирусный формат** (select) — Секретная фраза / Инструмент-открытие / Уволила специалиста / ChatGPT правда / Удали старое / Боль+решение / Лайфхак на миллион / Скетч / FOMO / 3 за 30 сек
- **Ссылка на пост** (url) — пустое при создании
- **Просмотры** (number) — пустое при создании
- **Заметки** (rich_text) — доп. заметки


### КЛЮЧЕВОЕ ПРАВИЛО:
Большие тексты (сценарии, черновики постов, статьи, карусели) хранятся ВНУТРИ страницы-карточки как content, а НЕ в свойствах. Свойства — только для тегирования и фильтрации.

## Определение режима работы

**Режим 1 — Полный автомат** (по умолчанию)
Пользователь загружает материал и просит «обработать контент» без уточнений.
→ Выполни ВСЕ этапы от анализа до генерации всех форматов.

**Режим 2 — Только анализ**
Пользователь просит «найти золотые куски» или «проанализировать материал».
→ Выполни только Этап 1 (анализ), покажи список золотых кусков, жди выбора.

**Режим 3 — Выборочная генерация**
Пользователь указывает конкретные куски или форматы: «сделай reels из куска 3», «только карусели».
→ Генерируй только запрошенное.

## Этап 0: Создание записи-исходника в Notion

Используй `notion-create-pages`:

```json
{
  "parent": {"data_source_id": "30eb40e2-0a6e-8051-aef1-000b33fa8801"},
  "pages": [{
    "properties": {
      "Name": "[краткое название материала]",
      "Тип": "[Пост/Транскрипция/Статья/Заметка/Идея]",
      "date:Дата:start": "[YYYY-MM-DD]",
      "date:Дата:is_datetime": 0,
      "Обработан": "__NO__"
    },
    "content": "[полный текст исходного материала]"
  }]
}
```

**Запомни URL созданной страницы** — он нужен для relation-связей.

## Этап 1: Анализ и вычленение золотых кусков

Проанализируй материал как эксперт по вирусному контенту. Выдели ВСЕ моменты с потенциалом.

### Что искать:
1. **Провокационные утверждения** — неожиданные мнения, разрушение стереотипов, смелые заявления
2. **Яркие истории и кейсы** — конкретные примеры, провалы и победы, трансформации «до/после»
3. **Эмоциональные триггеры** — боли и страхи, мечты и желания, фрустрации
4. **Неожиданные инсайты** — свежий взгляд, необычные связи, метафоры «взрывающие мозг»
5. **Конкретные механики** — работающие приёмы, пошаговые алгоритмы, лайфхаки
6. **Резонансные цитаты** — фразы что хочется сохранить, меткие формулировки

### Формат каждого куска:
```
### ЗОЛОТОЙ КУСОК №[номер]
**Тип:** [провокация/история/триггер/инсайт/механика/цитата]
**Суть:** [одним предложением]
**Исходный текст:** [точная цитата]
**Почему зацепит:** [какую эмоцию вызовет]
**Вирусный потенциал:** [высокий/средний — почему]
**Лучшие форматы:** [Reels/карусель/пост/статья/сторис]
```

Правила:
- Выделяй ВСЁ с потенциалом, даже если сомневаешься
- Копируй точные цитаты, не пересказывай
- Один кусок = одна завершённая мысль/история/идея
- Минимум 5, максимум 15 кусков

После анализа обнови страницу-исходник в Notion:
- Установи `"Золотые куски"` = число найденных кусков (`notion-update-page` → `update_properties`)
- Добавь полный анализ в контент страницы (`notion-update-page` → `insert_content_after`)

**В режиме 2:** покажи список кусков пользователю и спроси, какие обрабатывать.
**В режимах 1 и 3:** переходи к Этапу 2.

## Этап 2: Генерация контента → Notion

Каждая единица контента = **отдельная страница** в Notion-базе.

### Алгоритм для каждой карточки:

Используй `notion-create-pages`:

```json
{
  "parent": {"data_source_id": "[ID базы RU или EN]"},
  "pages": [{
    "properties": {
      "Название": "[название]",
      "Формат": "[Reels/Пост/Карусель/Тред/Лонгрид]",
      "Статус": "Черновик",
      "Тема": "[определи по содержанию]",
      "Исходник": "[URL страницы-исходника]",
      "Выложено": "__NO__"
    },
    "content": "[ПОЛНЫЙ ТЕКСТ контента — сценарий/пост/карусель/статья]"
  }]
}
```

**Для Reels** добавь ещё: `"Хронометраж"`, `"Вирусный формат"`, `"Кодовое слово"`.

**Можно создавать несколько страниц за один вызов** (массив pages).

### Формат 7: ЦИТАТНИК
Свойства: Формат=Пост, Заметки=«цитатник». Контент: 5-10 цитат из исходника с коротким контекстом к каждой.

## Этап 3: Финализация в Notion

1. **Обнови исходник** — `"Обработан"` → `"__YES__"` через `notion-update-page` → `update_properties`
2. **Проверь связи** — все карточки связаны с исходником
3. **Покажи сводку:** сколько карточек, в какой базе, ссылка на Content System

## Этап 4: Контроль качества

Проверь КАЖДЫЙ контент:
- Цепляет ли с первых слов?
- Есть ли конкретика (цифры, названия, примеры)?
- Вызывает ли эмоции?
- Не шаблон ли?
- Есть ли CTA?
- **Текст ПЕРЕПИСАН в вирусном стиле или остались формулировки из исходника?**
- **Прочитай вслух — звучит как живая речь или как статья?**
- **Нет ли канцеляризмов (является, позволяет, данный, в рамках)?**

Красные флаги — **переделывать немедленно:**
- Вода, общие слова, шаблоны, нет эмоций
- Текст звучит как пересказ исходника, а не как самостоятельный контент
- Фразы длиннее 15 слов (для постов) или 10 слов (для Reels)
- Канцеляризмы и «экспертный» тон вместо живого разговорного

## Важные правила стиля

**Главное правило: ты НЕ пересказываешь исходник — ты ПЕРЕРОЖДАЕШЬ его.**

Исходный текст = сырьё для смыслов и фактов. Финальный текст = как будто ты рассказываешь это в сторис или голосовым сообщением подруге. Если фраза звучит как из учебника, статьи или экспертного поста — перепиши до тех пор, пока не зазвучит как живая речь.

**Конкретные правила:**
- По умолчанию русский язык
- Живой разговорный тон, как с подругой/другом — с эмоциями, паузами, восклицаниями
- «ты», не «вы»
- Эмоции важнее правильности
- Конкретика бьёт абстракцию
- Каждый текст начинается с самого сильного элемента
- Фразы короткие — максимум 12-15 слов, чаще 5-8

**Запрещённые слова и обороты** (заменяй на живые аналоги):
- «является» → «это»
- «позволяет» → «можно» / «даёт»
- «в рамках» → убери совсем
- «данный» → «этот» / убери
- «осуществить» → «сделать»
- «обеспечивает» → «даёт» / «делает так, что»
- «необходимо» → «нужно» / «надо»
- «представляет собой» → «это»
- «на сегодняшний день» → «сейчас»
- Любые причастные обороты длиннее 5 слов — перепиши в короткие фразы

**Тест на живость:** прочитай текст вслух. Если где-то спотыкаешься или звучишь как робот — перепиши это место.
