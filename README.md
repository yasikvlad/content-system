# Content System — Контент-система

Плагин для автоматической генерации вирусного маркетингового контента из исходного материала.

## Команды

- `/reels [текст]` — 6 сценариев Reels (4 коротких + 2 длинных) с байтами для Telegram
- `/short-post [текст]` — 3-7 коротких постов (до 1000 знаков) по стилю примеров-референсов
- `/long-post [текст]` — длинный пост (2000-3000 знаков) по одному из 5 структурных паттернов
- `/carousel [текст]` — карусель для Instagram по одному из 6 типов
- `/article [текст]` — статья по жанровому подходу (4 жанра + 12 принципов)
- `/content [текст]` — полный конвейер: анализ → золотые куски → все форматы

## Скиллы

### reels-creator
Создатель вирусных Reels с 10 форматами, встроенными референсами из ниши AI/маркетинг, 5 паттернами хуков и системой байтов для Telegram.

### short-post-creator
Создатель вирусных коротких постов (до 1000 знаков). Анализирует 9 реальных постов-референсов как образцы стиля, ритма и подачи. Адаптация под Threads, Instagram, Telegram, LinkedIn, Twitter.

### long-post-creator
Создатель длинных постов (2000-3000 знаков). Содержит 5 структурных паттернов (история с трансформацией, механика, разрушение стереотипа, итоги, наблюдение за трендом), 12 принципов живого поста и платформенную адаптацию.

### carousel-post-creator
Создатель Instagram-каруселей. Содержит 6 типов каруселей (пошаговая инструкция, список ошибок, инсайт с доказательствами, до/после, сравнение/VS, история-кейс), анатомию идеального слайда и структуру текста под постом.

### article-creator
Создатель качественных статей. Содержит 4 редакторских жанра (практическое руководство, притча с механикой, философское эссе, консультационный текст), 12 универсальных принципов копирайтинга, 3 детальных редакторских разбора реальных статей.

### content-pipeline
Полный конвейер: анализ исходника → выделение золотых кусков → генерация всех форматов. Делегирует специализированным скиллам (reels-creator, short-post-creator, long-post-creator, carousel-post-creator, article-creator) для максимального качества.

## Использование

Вставь текст поста, транскрипции или статьи после команды — и получи готовый контент.
